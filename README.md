# PythonFileSorter
Python Script that sorts dowloaded files into different folders based on file extensions.

## How To Setup

## Features List

- [ ] Automatic launch at Windows Startup
- [ ] Running in background
  


Categories (destination system folders): 

- [ ] Documents 
  - .pdf
  - .txt
  - .doc & .docx & .odt
  - .xls & .xlsx
  - .ppt & .pptx

- [ ] Videos
  - .mp4
  - .avi
  - .m4v
  - .mkv
  - .mpg & .mpeg
  - .wmv
  - .mov
  - .h264


- [ ] Images
  - .jpg & .jpeg
  - .png
  - .gif

- [ ] Music
  - .mp3 
  - .mpa
  - .wav & .wma

- [ ] Left in Dowloads
  - .zip & .7z & .rar
  - .exe
  - .msi
  - everything else
